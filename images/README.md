# images/

Сюда кладутся фотографии сайта. Имена файлов **фиксированы** — `index.html`
ссылается именно на них. Положили файл с нужным именем — фото появилось на странице.
Файла нет — на его месте показывается заглушка с описанием, страница не ломается.

## Требования

- Формат: **JPG**. Хотите WebP/PNG — поменяйте `src` соответствующего `<img>` в `index.html`.
- Ширина: от 1000 px, лучше 1400–1600 px.
- Фото кадрируется по `object-fit: cover` — точные пропорции некритичны,
  но оставляйте запас по краям, центр не должен «обрезаться».
- `team-olga.jpg` — только реальное фото Olga Duma. Сток для портрета не использовать.

## Где брать картинки

1. **Реальные фото событий YBC** — лучший вариант. Раздел event-reports на ybc.club,
   Instagram и Facebook филиалов сети. Для галереи и программы — только так.
2. **Бесплатный сток** — pexels.com, unsplash.com, pixabay.com. Свободно для
   коммерческого использования на сайте, без оплаты и без указания автора.
