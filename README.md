# YBC Benelux — лендинг

Прототип посадочной страницы филиала **Young Business Club Benelux** (Люксембург).
Одностраничный сайт: членство в клубе + анонс официального открытия филиала.

> Статус: **рабочий прототип**. Структура и дизайн готовы, часть контента — заглушки
> и черновики (см. раздел «Что нужно доделать»).

---

## Как запустить

Зависимостей нет — это один HTML-файл.

- **Просто открыть:** дважды кликнуть `index.html` — откроется в браузере.
- **Локальный сервер** (рекомендуется, чтобы корректно работали якорные ссылки):
  ```bash
  python3 -m http.server 8000
  # затем открыть http://localhost:8000
  ```

## Стек

Чистый HTML + CSS + JavaScript в одном файле `index.html`.
Без фреймворков, без CDN, без сборки. Вёрстка **mobile-first**:
адаптив через `clamp()` и CSS Grid с `auto-fill`/`auto-fit`.

## Двуязычность (UA / EN)

Сайт двуязычный: украинский (по умолчанию) и английский. Русский не используется.

**Как устроено:** каждый переводимый элемент несёт атрибут `data-i18n="ключ"`
(или `data-i18n-ph="ключ"` для placeholder инпутов). Все тексты лежат в JS-объекте
`I18N = { ua: {...}, en: {...} }` в конце файла. Функция `setLang(lang)`
проходит по элементам и подставляет нужный перевод. Переключатель — кнопки в шапке.

**Чтобы изменить текст:** найти ключ в объекте `I18N` — править там, а не в HTML.
HTML-содержимое — это только запасной вариант до запуска JS.

**Чтобы добавить язык:** добавить новый объект (напр. `fr: {...}`) в `I18N`
и кнопку с `data-lang="fr"` в блок `.lang` в шапке.
